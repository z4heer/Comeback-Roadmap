# 02 - Flagship Project

Purpose
- Build a small but realistic backend service that demonstrates architecture, testing, security, and deployment.

Structure
- docs/          -> architecture.md, api-design.md, decisions.md
- backend/       -> application code (FastAPI)
- database/      -> schema, migrations
- tests/         -> unit + integration tests

How to contribute
- Work from feature branches named: feat/<short-desc>
- Open PRs against main with description & checklist
